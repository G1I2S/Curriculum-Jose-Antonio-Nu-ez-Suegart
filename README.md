## Descripción

Curriculum vitae profesional interactivo desarrollado con tecnologías web modernas. Este proyecto presenta mi trayectoria profesional, formación académica, habilidades y experiencia laboral de forma atractiva y accesible.

## Características

- **Diseño Responsivo**: Adaptable a diferentes dispositivos (móvil, tablet, desktop)
- **Navegación Intuitiva**: Interfaz limpia y fácil de usar
- **Portafolio Integrado**: Galería de proyectos anteriores con capturas de pantalla
- **Enlaces Directos**: Acceso rápido a LinkedIn y GitHub
- **Búsqueda**: Funcionalidad de búsqueda en la barra de navegación

## Tecnologías Utilizadas

- **HTML5**: Estructura semántica del contenido
- **CSS3**: Estilos personalizados
- **Bootstrap 5.3.8**: Framework CSS para diseño responsivo
- **Git/GitHub**: Control de versiones

## Instalación y Uso

1. Clonar el repositorio:
```bash
git clone https://github.com/iLmUcCo/Curriculum-Jose-Antonio-Nu-ez-Suegart.git
```

2. Navegar al directorio del proyecto:
```bash
cd Curriculum-Jose-Antonio-Nu-ez-Suegart
```

3. Abrir `index.html` en tu navegador preferido o usar un servidor local:
```bash
# Con Python 3
python -m http.server 8000

# Con Node.js (npx)
npx serve
```

4. Visitar `http://localhost:8000` en tu navegador
